# Educator Portfolio

Personalized portfolio template for Alex Kong'a (Math & Computer Science Teacher).

Preview:

1. Open `index.html` in your browser from this folder.

Deploy to GitHub Pages (recommended):

1. Create a GitHub repository (e.g., `username/portfolio`).
2. Push this folder to the repository's `main` branch.
3. Enable GitHub Pages from the repository settings or use the provided GitHub Actions workflow included in `.github/workflows/deploy.yml`.
